# Employee-Attrition-Analysis
Employee Attrition Analysis is a comprehensive data analytics project designed to identify the key factors that contribute to employee turnover. By analyzing demographic data, job satisfaction metrics, and performance indicators, this project provides actionable insights to help HR departments improve retention strategies.
